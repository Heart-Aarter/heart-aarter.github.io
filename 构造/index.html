构造

CF1039A Timetable

题目分析

已知发车时刻序列 \(a\) 严格递增。我们需要构造一个到达时刻序列 \(b\)，序列 \(b\) 也必须严格递增。

首先，这个序列必须合法。什么是合法序列？ 在这题里面，合法的 \(b\) 序列必须满足： \[b_{\text{到达顺序}} \ge a_{\text{出发顺序}} + t\]

最简单的合法 \(b\) 序列就是 \(b_i = a_i + t\)，即所有的公交车按原顺序依次到达 B 站。

因为 \(x_i\) 是第 \(i\) 辆出发的公交车最晚到达顺序，所以如果符合题意的 \(b\) 序列能够被构造出来，\(x_i \ge i\) 应该是成立的，因为 \(b_i = a_i + t\) 这个保底的序列一定合法。 所以一旦发现 \(x_i < i\)，就可以判断无法构造成功了，直接输出 No。

构造逻辑推导

再来看如何具体构造。 设初始的序列 \(b\) 是 \(b_i = a_i + t\) 这样的序列。 假设第 \(i\) 辆出发的公交车最晚是第 \(j = x_i\) 个到达的，不能是第 \(j+1\) 个到达的了。

为什么这个公交车不能是第 \(j+1\) 个到达的车了？ 不是因为它不能接着在路上墨迹了，而是因为第 \(j+1\) 辆车没有办法赶在它之前到了，也就是那堵拦截的“墙”： \[b_j < a_{j+1} + t\]

为什么这个公交车可以是第 \(j\) 个到达的车？ 是因为第 \(i+1\) 到第 \(j\) 辆出发的车可以在它之前到达 B 站，这很明显，因为： \[b_{i+1} > b_i \ge a_i + t\] 所以，第 \(i+1\) 到第 \(j\) 辆出发的车都可以往前挪一位，填补因为第 \(i\) 个出发的车选择在第 \(j\) 个到达所产生的空缺。

显然可知，第 \(i+1\) 到第 \(j\) 辆出发的车，最晚到达顺序也是第 \(j\) 个。记 \(k\) 满足 \(i+1 \le k \le j\)，那么第 \(k\) 到第 \(j\) 辆出发的车都同样可以往前挪一位填补空缺；同时，第 \(k\) 辆出发的车也没办法成为第 \(j+1\) 辆到达的车，因为同样被 \(b_j < a_{j+1} + t\) 挡住了。

故第 \(i\) 辆出发的车到第 \(x_i\) 辆出发的所有车，它们最晚的到达顺序都是 \(x_i\)，我们给它起个名字叫“相同区间”。
