CF1039A Timetable

题目分析

已知发车时刻序列 \(a\) 严格递增。我们需要构造一个到达时刻序列 \(b\)，序列 \(b\) 也必须严格递增。

首先，这个序列必须合法。什么是合法序列？ 在这题里面，合法的 \(b\) 序列必须满足： \[b_{\text{到达顺序}} \ge a_{\text{出发顺序}} + t\]

最简单的合法 \(b\) 序列就是 \(b_i = a_i + t\)，即所有的公交车按原顺序依次到达 B 站。

因为 \(x_i\) 是第 \(i\) 辆出发的公交车最晚到达顺序，所以如果符合题意的 \(b\) 序列能够被构造出来，\(x_i \ge i\) 应该是成立的，因为 \(b_i = a_i + t\) 这个保底的序列一定合法。 所以一旦发现 \(x_i < i\)，就可以判断无法构造成功了，直接输出 No。

构造逻辑推导

再来看如何具体构造。 设初始的序列 \(b\) 是 \(b_i = a_i + t\) 这样的序列。 假设第 \(i\) 辆出发的公交车最晚是第 \(j = x_i\) 个到达的，不能是第 \(j+1\) 个到达的了。

为什么这个公交车不能是第 \(j+1\) 个到达的车了？ 不是因为它不能接着在路上墨迹了，而是因为第 \(j+1\) 辆车没有办法赶在它之前到了，也就是那堵拦截的“墙”： \[b_j < a_{j+1} + t\]

为什么这个公交车可以是第 \(j\) 个到达的车？ 是因为第 \(i+1\) 到第 \(j\) 辆出发的车可以在它之前到达 B 站，这很明显，因为： \[b_{i+1} > b_i \ge a_i + t\] 所以，第 \(i+1\) 到第 \(j\) 辆出发的车都可以往前挪一位，填补因为第 \(i\) 个出发的车选择在第 \(j\) 个到达所产生的空缺。

显然可知，第 \(i+1\) 到第 \(j\) 辆出发的车，最晚到达顺序也是第 \(j\) 个。记 \(k\) 满足 \(i+1 \le k \le j\)，那么第 \(k\) 到第 \(j\) 辆出发的车都同样可以往前挪一位填补空缺；同时，第 \(k\) 辆出发的车也没办法成为第 \(j+1\) 辆到达的车，因为同样被 \(b_j < a_{j+1} + t\) 挡住了。

故第 \(i\) 辆出发的车到第 \(x_i\) 辆出发的所有车，它们最晚的到达顺序都是 \(x_i\)，我们给它起个名字叫“相同区间”。

二叉树

二叉树

定义：由根结点和左右子树组成的一种数据结构，左右子树可以为空，且左右子树本身也是二叉树。

根结点：没有父亲的结点叫做二叉树的根结点。

树的深度：从根结点到最深叶子结点，最多经过的层数叫做二叉树的深度。

叶子结点：如果一个结点没有任何子树，那么这个结点就是叶子结点。

完美二叉树：如果一个二叉树的所有叶子结点都在同一层，且除叶子结点外每个结点都有两个子结点，那么这个二叉树称为完美二叉树。

完全二叉树：如果一个二叉树只有最后一层不是满的，而且最后一层的结点是从左到右连续排列的，那么这个二叉树称为完全二叉树。

Dijkstra算法

Dijkstra算法

用处：用于求单源最短路，适用于边权非负的有向图或无向图。

实现过程概述：从起点 s 出发，维护数组 dis[i]，表示当前已知的从 s 到 i 的最短距离估计值。每次在所有尚未确定最短路的点中，选取 dis 最小的点，将其最短路确定下来，再用它去更新相邻点的距离。

实现原理：贪心。每次都优先确定当前距离源点最近的未确定点。

实现原理证明过程

前提条件：图中所有边权均为非负值。

设源点为 s。

记号说明：

  • dis[i] 表示当前算法中，从 s 到 i 的最短距离估计值。
  • pd[i] 表示点 i 的最短路是否已经被确定。

初始化：

  • dis[s]=0
  • 对于 i!=s，dis[i]=INF
  • 初始时所有 pd[i]=0

不变量：

  • 对于所有满足 pd[i]=1 的点 i，dis[i] 已经等于从 s 到 i 的真实最短路长度；
  • 对于所有满足 pd[i]=0 的点 i，dis[i] 是当前已知的从 s 到 i 的最短距离估计值。

需证明：

在上述前提下，Dijkstra 算法能够正确求出源点 s 到所有可达点的最短路长度；对于不可达点，其 dis 值最终保持为 INF。

证明：

设 \(S={i \mid pd[i]=1}\)，即当前已经确定最短路的点集。

使用数学归纳法证明。

初始化时，dis[s]=0，其余点的 dis 都为 INF，因此第一次被选出的点一定是 s。显然，dis[s]=0 就是从 s 到 s 的真实最短路长度，所以不变量成立。

假设某一轮开始前，不变量成立。此时从所有 \(v \notin S\) 中选取 dis[v] 最小的点 x，下面证明 dis[x] 就是从 s 到 x 的真实最短路长度。

假设存在一条从 s 到 x 的路径，其长度小于 dis[x]。在这条路径上，从前往后找到第一个不属于 S 的点，记为 y，设 y 的前一个点为 z。由于 y 是这条路径上第一个不属于 S 的点，因此 z 一定属于 S。

根据归纳假设，dis[z] 已经等于从 s 到 z 的真实最短路长度。当 z 被加入 S 时，算法一定会用 z 去松弛与它相邻的点，因此有：\(dis[y] \le dis[z] + w(z,y)\)

而 dis[z] + w(z,y) 正好等于这条路径到达 y 时的长度。又因为图中边权非负，所以这条路径到达 y 的长度不大于到达 x 的长度，从而有：\(dis[y] < dis[x]\)

但这与 x 是所有 \(v \notin S\) 中 dis[v] 最小的点矛盾。

因此假设不成立，dis[x] 必然等于从 s 到 x 的真实最短路长度。

将 x 加入 S 后，再用 x 去松弛与它相邻的点，就把所有经过 x 的可能更短路径也纳入了考虑，因此不变量继续成立。

所以，由数学归纳法可知，Dijkstra 算法能够正确求出源点 s 到所有可达点的最短路长度；对于不可达点，其 dis 值始终保持为 INF。
